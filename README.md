# Домашнее задание: Работа с датами

## Задачи

### 1. Калькулятор возраста (age_calc.py)
- Запрашивает дату рождения
- Вычисляет возраст в годах, месяцах и днях
- Показывает дней до следующего дня рождения

### 2. Планировщик задач (task_planner.py)
- Хранит список задач с датой выполнения
- Показывает задачи на сегодня
- Показывает просроченные задачи
- Показывает задачи на ближайшие 7 дней

### 3. Форматирование дат (date_formatter.py)
- `"10 апреля 2026 года"` — русский формат
- `"10.04.2026"` — формат с точками
- `"2026-04-10"` — ISO формат
- `"Apr 10, 2026"` — английский формат

## Запуск

```bash
python age_calc.py
python task_planner.py
python date_formatter.py
```
